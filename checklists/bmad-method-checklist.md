# BMAD Method Checklist

## Purpose

This checklist is used to review whether the BMAD Method is clearly applied in the AI-in-TVET Microcredential project.

BMAD is used in this project as a structured workflow:

- Build the idea
- Manage the content
- Assess the learning
- Deliver the output

## B: Build the Idea

Check whether the project idea is clear.

- [ ] The project topic is clearly defined.
- [ ] The target group is identified.
- [ ] The main aim is explained.
- [ ] The learning need or problem is described.
- [ ] The project is connected to AI-in-TVET.
- [ ] The project is relevant for teaching, training, or learning.
- [ ] The expected final output is described.

## M: Manage the Content

Check whether the content is organized clearly.

- [ ] The folder structure is complete.
- [ ] README.md explains the project.
- [ ] Course overview is included.
- [ ] Learning outcomes are included.
- [ ] Module files are included.
- [ ] Task files are included.
- [ ] Templates are included.
- [ ] Checklists are included.
- [ ] Files have clear names.
- [ ] Content is easy to follow.

## A: Assess the Learning

Check whether learning assessment is clear.

- [ ] Assessment strategy is included.
- [ ] Assessment criteria are transparent.
- [ ] Module tasks are included.
- [ ] Final project task is included.
- [ ] Evidence submission is explained.
- [ ] Passing requirement is included.
- [ ] Assessment rubric is included.
- [ ] Feedback process is included.
- [ ] Assessment is connected to learning outcomes.

## D: Deliver the Output

Check whether the final output can be delivered.

- [ ] Online delivery is explained.
- [ ] Offline delivery is explained.
- [ ] GitHub repository can be used for sharing.
- [ ] LiaScript or GitHub Pages can be used if required.
- [ ] Offline folder or ZIP package can be created.
- [ ] Certificate or badge template is included.
- [ ] Final project evidence is included.
- [ ] The project can be shared with learners or reviewers.

## AI Support Checklist

Check whether AI support is used responsibly.

- [ ] AI is used to support content creation.
- [ ] AI-generated content is reviewed.
- [ ] AI-generated content is improved manually.
- [ ] AI use is mentioned in the reflection.
- [ ] Final content is checked for clarity and accuracy.
- [ ] AI does not replace learner understanding.
