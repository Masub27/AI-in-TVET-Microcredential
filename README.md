# AI-in-TVET Microcredential

## Project Overview
This project develops a micro-credential for AI-supported teaching workflow creation in TVET. It focuses on using GitHub, VS Code, GitHub Copilot, LiaScript, and the BMAD Method to create online and offline learning materials.

## Content Basis
This project is based on the AI-in-TVET content and the teaching workflow structure shared by Dr. Hannes Tegelbeckers. It also follows UNESCO and European Union guidance on micro-credentials.

## Aim
The aim is to design a structured micro-credential that helps learners create, manage, and publish AI-supported educational content for online and offline use.

## Target Group
- TVET students
- Teachers
- Trainers
- Educational technology beginners
- Workshop participants

## Tools Used
- GitHub
- VS Code
- GitHub Copilot
- LiaScript
- BMAD Method

## Learning Outcomes
After completing this micro-credential, learners will be able to:

1. Explain the concept of micro-credentials.
2. Set up a GitHub and VS Code teaching workflow.
3. Use the BMAD Method for structured course creation.
4. Create AI-supported learning materials.
5. Develop online and offline teaching outputs.
6. Submit a final teaching workflow project.

## Project Structure
- `assets/` contains images, audio, videos, and icons.
- `checklists/` contains quality and workflow checklists.
- `data/` contains learner progress and assessment data.
- `docs/` contains course documentation and standards.
- `examples/` contains sample learning materials and sample outputs.
- `materials/` contains the main learning modules.
- `Outputs/` contains certificates, badges, reports, and offline packages.
- `skeletons/` contains starter structures.
- `tasks/` contains learner tasks.
- `templates/` contains reusable templates.

## Course Modules
1. Introduction to Micro-Credentials
2. GitHub and VS Code Setup
3. BMAD Method for Course Design
4. AI-Supported Content Generation
5. Online and Offline Course Package
6. Final Project

## Assessment
- Module tasks: 30%
- Online/offline teaching package: 30%
- Final project: 40%

## Final Output
The final output is a complete AI-supported online/offline teaching workflow package with learning materials, tasks, assessment, and certificate template.
