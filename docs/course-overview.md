# Course Overview

## Title
AI-in-TVET Microcredential

## Short Description
This micro-credential introduces learners to AI-supported teaching workflow creation in Technical and Vocational Education and Training (TVET). It focuses on using GitHub, VS Code, GitHub Copilot, LiaScript, and the BMAD Method to create online and offline educational materials.

## Aim
The aim of this micro-credential is to help learners design, organize, and publish AI-supported teaching materials using a structured digital workflow.

## Target Group
This micro-credential is designed for:

- TVET students
- Teachers
- Trainers
- Educational technology beginners
- Workshop participants

## Duration
4 weeks

## Estimated Workload
The estimated workload is 20 to 30 hours.

Example distribution:

| Activity | Estimated Time |
|---|---:|
| Reading and preparation | 5 hours |
| GitHub and VS Code setup | 4 hours |
| BMAD workflow practice | 5 hours |
| AI-supported content creation | 6 hours |
| Final project | 8 to 10 hours |

## Mode of Learning
This micro-credential supports:

- Online learning through GitHub and LiaScript
- Offline learning through downloaded course files
- Self-directed learning
- Practical project work

## Tools
Learners will use:

- GitHub
- VS Code
- GitHub Copilot
- LiaScript
- BMAD Method

## Course Modules
The course contains six modules:

1. Introduction to Micro-Credentials
2. GitHub and VS Code Setup
3. BMAD Method for Course Design
4. AI-Supported Content Generation
5. Online and Offline Course Package
6. Final Project

## Main Learning Output
By the end of this micro-credential, learners will create a complete AI-supported online/offline teaching workflow package.

The final package should include:

1. README file
2. Course overview
3. Learning outcomes
4. Learning modules
5. Learner tasks
6. Assessment strategy
7. Quality checklist
8. Certificate or badge template
9. Online/offline delivery plan

## Assessment
Learners will be assessed through:

- Module tasks
- Practical evidence
- Final teaching workflow project

## Completion Requirement
Learners must complete all required tasks and submit a final project. A minimum achievement level of 60% is required to complete the micro-credential.

## Final Credential
After successful completion, learners can receive a certificate or badge showing that they completed the AI-in-TVET Microcredential.
