# Module 4: AI-Supported Content Generation

## Module Overview

This module introduces how AI tools can support the creation of educational content for TVET.

AI can help teachers, trainers, and learners create lesson plans, learning activities, assessment tasks, examples, explanations, and feedback. However, AI-generated content must always be reviewed, corrected, and improved by humans.

## Module Aim

The aim of this module is to help learners use AI support responsibly to create and improve teaching materials.

## Learning Outcomes

After completing this module, learners will be able to:

1. Explain how AI can support educational content creation.
2. Use AI prompts to generate learning materials.
3. Review and improve AI-generated content.
4. Create a short AI-supported learning activity.
5. Connect AI-generated content with learning outcomes and assessment.
6. Reflect on the strengths and limitations of AI tools.

## Key Concepts

### AI-Supported Content Generation

AI-supported content generation means using AI tools to help create learning materials.

AI can help with:

- Lesson planning
- Module outlines
- Learning outcomes
- Activities
- Quiz questions
- Assessment rubrics
- Explanations
- Examples
- Feedback text
- Summaries

### Human Review

AI output is not always correct. A teacher, trainer, or learner must check the content before using it.

Human review should check:

- Accuracy
- Clarity
- Relevance
- Language level
- Bias
- Copyright or source issues
- Connection with learning outcomes
- Suitability for the target group

### Prompting

A prompt is an instruction given to an AI tool.

A good prompt should include:

- Topic
- Target group
- Learning level
- Required output
- Format
- Length
- Language
- Assessment requirement

Example prompt:

Create a beginner-level learning activity for TVET students about using GitHub for storing teaching materials. Include aim, steps, task, and assessment criteria.

## AI Tools in This Project

Learners may use:

- GitHub Copilot
- ChatGPT or another AI assistant
- VS Code AI extensions
- Other approved AI-supported writing tools

## Responsible Use of AI

AI should support learning, not replace learner thinking.

Learners should:

- Use AI as a helper.
- Check all AI-generated content.
- Improve the output manually.
- Mention when AI was used.
- Avoid copying without understanding.
- Make sure the final work is relevant and accurate.

## Step 1: Choose a Teaching Topic

Choose one small topic related to TVET or digital teaching.

Example topics:

- Introduction to GitHub
- Creating a Markdown file
- Using AI for lesson planning
- Online and offline learning materials
- Basic safety training
- Digital assessment in TVET

## Step 2: Write a Prompt

Write a clear AI prompt.

Your prompt should include:

1. Topic
2. Target group
3. Learning level
4. Type of material
5. Required sections

Example:

Create a short learning activity for beginner TVET students about Markdown. Include module aim, learning outcomes, activity steps, task, evidence, and assessment criteria.

## Step 3: Generate Content

Use an AI tool to generate the first version of the learning material.

Do not submit the AI output directly without checking it.

## Step 4: Review and Improve

Read the AI-generated content carefully.

Improve:

- Grammar
- Clarity
- Structure
- Accuracy
- Examples
- Assessment criteria
- Connection with learning outcomes

## Step 5: Save the Final Version

Save the improved learning material as a Markdown file.

Example file name:

ai-supported-learning-activity.md

## Learning Activity

Use an AI tool to create one short learning activity for a TVET or digital teaching topic.

Then improve the activity manually.

## Task

Create one AI-supported learning activity with the following sections:

1. Title
2. Target group
3. Aim
4. Learning outcomes
5. Learning activity
6. Learner task
7. Evidence to submit
8. Assessment criteria
9. Short reflection on how AI was used

## Evidence to Submit

Submit:

1. Your original AI prompt.
2. The first AI-generated version.
3. Your improved final version.
4. A short reflection of 100–150 words.

## Reflection Questions

Answer these questions in your reflection:

- What did AI help you create?
- What did you change or improve?
- Was the AI output fully correct?
- How can AI support teachers and trainers in TVET?
- What are the limitations of using AI?

## Assessment Criteria

| Criteria | Description |
|---|---|
| Prompt Quality | The prompt is clear and specific. |
| AI Output | The AI-generated content is relevant to the topic. |
| Human Improvement | The learner improves the AI output manually. |
| Learning Design | The final activity includes aim, outcomes, task, and assessment. |
| Reflection | The learner explains how AI was used and improved. |
| Relevance | The content is suitable for TVET or digital teaching. |
