# Module 3: BMAD Method for Course Design

## Module Overview

This module introduces the BMAD Method as a structured workflow for planning and creating educational content.

In this micro-credential, the BMAD Method is used to guide the creation of AI-supported teaching materials for online and offline learning.

## Module Aim

The aim of this module is to help learners use the BMAD Method to plan a small teaching module in a clear and organized way.

## Learning Outcomes

After completing this module, learners will be able to:

1. Explain the purpose of the BMAD Method.
2. Use BMAD to plan learning content.
3. Define a target group for a teaching module.
4. Write clear learning outcomes.
5. Create learning activities and tasks.
6. Connect assessment with learning outcomes.

## What Is the BMAD Method?

In this project, BMAD is used as a structured design process for educational content creation.

BMAD can be understood as:

- B: Build the idea
- M: Manage the content
- A: Assess the learning
- D: Deliver the output

This means learners first define the course idea, then organize the content, create assessment tasks, and finally prepare the online or offline learning output.

## Step 1: Build the Idea

In this step, learners define the basic idea of the teaching module.

Important questions:

- What is the topic?
- Who is the target group?
- Why is this topic important?
- What problem does the learning module solve?
- What should learners be able to do after completing it?

Example:

Topic: Creating AI-supported lesson materials  
Target group: TVET teachers and students  
Main aim: Learners create one digital learning activity using AI support.

## Step 2: Manage the Content

In this step, learners organize the content into clear sections.

A simple module can include:

- Module overview
- Aim
- Learning outcomes
- Key concepts
- Learning activity
- Task
- Evidence to submit
- Assessment criteria
- Completion checklist

Good content should be:

- Clear
- Short
- Practical
- Connected to the learning outcomes
- Easy to use online and offline

## Step 3: Assess the Learning

In this step, learners design assessment tasks.

Assessment should check whether the learning outcomes were achieved.

Example assessment tasks:

- Write a short reflection.
- Create a Markdown learning file.
- Upload a GitHub repository.
- Submit screenshots as evidence.
- Present a final teaching workflow package.

Assessment criteria should be transparent so learners know what is expected.

## Step 4: Deliver the Output

In this step, learners prepare the final output.

The output can be:

- GitHub repository
- LiaScript course page
- Offline folder
- PDF or Markdown materials
- Certificate or badge template
- Final project package

The output should work for both online and offline learning where possible.

## BMAD Workflow Table

| BMAD Step | Meaning | Example in This Project |
|---|---|---|
| Build | Define the idea and purpose | Select a TVET teaching topic |
| Manage | Organize the learning content | Create modules, tasks, and templates |
| Assess | Check learner achievement | Use tasks, evidence, and rubric |
| Deliver | Publish or share the output | GitHub, LiaScript, and offline package |

## Learning Activity

Choose one small TVET topic and plan it using the BMAD Method.

Example topics:

- AI-supported lesson planning
- GitHub for teaching materials
- Digital assessment in TVET
- Online and offline course design
- LiaScript for interactive learning

## Task

Create a short BMAD course plan with the following sections:

1. Title of the module
2. Target group
3. Main aim
4. Three learning outcomes
5. Learning activity
6. Assessment task
7. Evidence to submit
8. Delivery format

## Evidence to Submit

Submit one of the following:

- Markdown file with your BMAD course plan.
- Screenshot of the BMAD plan in VS Code.
- GitHub link to the BMAD plan.
- Offline project folder containing the plan.

| Criteria | Description |
|---|---|
