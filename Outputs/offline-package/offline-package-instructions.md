# Offline Package Instructions

## Purpose
This file explains how to prepare the offline version of the AI-in-TVET Microcredential.

## Steps

1. Save all project files.
2. Check that README.md is complete.
3. Check that all module files are included.
4. Check that task files are included.
5. Check that templates and checklists are included.
6. Right-click the project folder.
7. Select Compress or Create ZIP.
8. Rename the ZIP file.

## Suggested ZIP Name
AI-in-TVET-Microcredential-offline-package.zip

## Test
Before submission, open the ZIP file and check that all files are available without internet.
