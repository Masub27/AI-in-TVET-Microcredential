# Task 2: Open Project in VS Code

## Task Overview

In this task, learners open the AI-in-TVET Microcredential project folder in VS Code.

VS Code is used to create, edit, and organize project files such as Markdown documents, modules, tasks, templates, and checklists.

## Aim

The aim of this task is to help learners use VS Code as the main workspace for managing the micro-credential project.

## Instructions

Follow these steps:

1. Open VS Code.
2. Click File.
3. Click Open Folder.
4. Select the project folder.
5. Check that the folder opens in the Explorer panel.
6. Make sure the main folders are visible.
7. Open README.md.
8. Edit or review the README file.
9. Save the file.

## Required Folder Structure

The project folder should include:

- assets/
- checklists/
- data/
- docs/
- examples/
- materials/
- Outputs/
- skeletons/
- tasks/
- templates/
- README.md

## Expected Output

At the end of this task, the learner should have:

- The project opened in VS Code.
- Folder structure visible in the Explorer panel.
- README.md opened and checked.
- Files ready for editing.

## Evidence to Submit

Submit one of the following:

- Screenshot of the project folder opened in VS Code.
- Screenshot of README.md opened in VS Code.
- Short note confirming that the folder was opened successfully.
