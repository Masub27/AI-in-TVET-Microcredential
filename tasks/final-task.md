# Final Task: Submit the AI-in-TVET Microcredential Project

## Task Overview

This is the final task of the AI-in-TVET Microcredential.

Learners must submit a complete AI-supported online/offline teaching workflow package. The final project should show that the learner can use GitHub, VS Code, AI support, LiaScript, and the BMAD Method in a structured educational workflow.

## Aim

The aim of this final task is to assess whether learners can create and submit a complete micro-credential project with documentation, learning materials, tasks, assessment, templates, and online/offline delivery.

## Final Project Requirement

Create a complete project folder or GitHub repository named:

AI-in-TVET-Microcredential

The final project must include:

- README.md
- docs/
- materials/
- tasks/
- templates/
- checklists/
- Outputs/

## Final Project Instructions

Follow these steps:

1. Review all documentation files.
2. Review all module files.
3. Review all task files.
4. Review all templates.
5. Review all checklists.
6. Complete the quality checklist.
7. Complete the BMAD checklist.
8. Complete the online/offline workflow checklist.
9. Prepare the final GitHub repository.
10. Prepare the offline ZIP package if required.
11. Write a final reflection.
12. Submit the final evidence.

## Final Reflection

Write a reflection of 200–300 words.

Answer these questions:

1. What did you create in this project?
2. How did you use the BMAD Method?
3. How did AI support your content creation?
4. What did you improve manually?
5. How can this project support TVET learning?
6. How can the project be used online and offline?
7. What did you learn from creating this micro-credential?

## Evidence to Submit

Submit:

- GitHub repository link
- Screenshot of the project in VS Code
- Screenshot of the GitHub repository
- Completed final reflection
- Completed quality checklist
- Offline ZIP package if required

## Assessment Criteria

| Criteria | Description | Percentage |
|---|---|---:|
| Project Structure | Folder and files are complete and organized. | 20% |
| Documentation | README, overview, outcomes, definition, standards, and assessment are complete. | 20% |
| Learning Materials | Modules are clear, useful, and connected to outcomes. | 20% |
| BMAD Application | BMAD Method is visible in planning, content, assessment, and delivery. | 15% |
| AI Use and Improvement | AI support is used responsibly and improved manually. | 10% |
| Online/Offline Delivery | Project can be used online and offline. | 10% |
| Reflection | Reflection explains learning and project development. | 5% |

## Passing Requirement

To pass, learners must achieve at least:

60% overall

The final project evidence must be submitted.
