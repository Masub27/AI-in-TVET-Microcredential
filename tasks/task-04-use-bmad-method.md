# Task 4: Use the BMAD Method

## Task Overview

In this task, learners use the BMAD Method to plan a small teaching workflow.

BMAD helps learners move step by step from an idea to a final learning output.

## Aim

The aim of this task is to help learners apply the BMAD Method for structured course or module design.

## BMAD Meaning

In this project, BMAD means:

- Build the idea
- Manage the content
- Assess the learning
- Deliver the output

## Instructions

Create a short BMAD plan for one learning topic.

Follow these steps:

1. Choose one TVET or digital teaching topic.
2. Define the target group.
3. Write the main aim.
4. Write three learning outcomes.
5. Organize the content into sections.
6. Create one learner activity.
7. Create one assessment task.
8. Explain how the output will be delivered online and offline.

## Suggested Topic Examples

You can choose one of these topics:

- AI-supported lesson planning
- GitHub for teaching materials
- VS Code for educational content
- LiaScript for online learning
- Online and offline course design
- Digital assessment in TVET

## BMAD Planning Template

Use this structure:

| BMAD Step | Your Plan |
|---|---|
| Build | Write the topic, target group, and main aim. |
| Manage | Write the content sections and learning activity. |
| Assess | Write the task, evidence, and assessment criteria. |
| Deliver | Explain online and offline delivery. |

## Expected Output

At the end of this task, the learner should have:

- One completed BMAD plan.
- Clear learning outcomes.
- One learning activity.
- One assessment task.
- Online/offline delivery explanation.

## Evidence to Submit

Submit one of the following:

- Markdown file with the BMAD plan.
- Screenshot of the BMAD plan in VS Code.
- GitHub link to the BMAD plan.

## Assessment Criteria

| Criteria | Description |
|---|---|
| Build | Topic, target group, and aim are clearly defined. |
| Manage | Content is organized into clear sections. |
| Assess | Task and assessment criteria are included. |
| Deliver | Online and offline delivery are explained. |
| Clarity | The plan is easy to understand. |
| Relevance | The topic is connected to AI-in-TVET or digital teaching. |

## Completion Checklist

Before submitting, check that:

- [ ] Topic is selected.
- [ ] Target group is defined.
- [ ] Main aim is written.
- [ ] Learning outcomes are written.
- [ ] Content sections are listed.
- [ ] Learning activity is included.
- [ ] Assessment task is included.
- [ ] Online delivery is explained.
- [ ] Offline delivery is explained.
- [ ] Evidence is ready for submission.

## Short Summary

This task helps learners apply the BMAD Method in a practical way. The learner plans a small teaching workflow by building the idea, managing content, assessing learning, and delivering the final output.
