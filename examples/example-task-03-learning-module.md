# Example Task 3: Learning Module

## Module Title
AI-Supported Lesson Planning in TVET

## Module Overview
This module helps TVET learners and teachers use AI tools to create a simple lesson activity.

## Aim
The aim is to create one AI-supported learning activity and improve it manually.

## Learning Outcomes
After completing this module, learners will be able to:

1. Write a clear AI prompt.
2. Generate a lesson activity using AI.
3. Review and improve AI-generated content.
4. Save the final activity as a Markdown file.

## Learning Activity
Learners write a prompt asking AI to create a lesson activity for a TVET topic.

## Task
Create one AI-supported lesson activity and improve it manually.

## Evidence to Submit
- Original AI prompt
- First AI-generated version
- Improved final version
- Short reflection

## Assessment Criteria
| Criteria | Description |
|---|---|
| Prompt | Prompt is clear and specific. |
| AI Output | Output is relevant to the topic. |
| Improvement | Learner improves the AI text manually. |
| Reflection | Learner explains how AI was used. |
