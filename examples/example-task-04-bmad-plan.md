# Example Task 4: BMAD Plan

## Topic
Using GitHub for Teaching Materials

## Build
Target group: TVET students and teachers

Aim: Learners will create a GitHub repository to store and share teaching materials.

Learning need: Teachers and learners need a simple way to organize online and offline learning materials.

## Manage
Content sections:

1. Introduction to GitHub
2. Creating a repository
3. Adding course files
4. Organizing folders
5. Sharing the repository link

## Assess
Assessment task:

Learners create a GitHub repository and upload a README file, one module, and one task.

Evidence:

- GitHub repository link
- Screenshot of repository
- Screenshot of VS Code folder

## Deliver
Online delivery:

The project is shared through GitHub.

Offline delivery:

The project folder is downloaded as a ZIP file and opened locally.
