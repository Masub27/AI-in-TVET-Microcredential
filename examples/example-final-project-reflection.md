# Example Final Project Reflection

In this project, I created an AI-supported online/offline teaching workflow package for TVET. I used the BMAD Method to build the idea, manage the content, assess the learning, and deliver the final output.

AI helped me create first drafts of learning materials, tasks, and assessment criteria. I reviewed and improved the content manually to make it clearer and more suitable for learners.

The project can support TVET learning because it gives learners practical tools for creating digital teaching materials. It can be used online through GitHub and offline as a downloaded folder.
