# Example BMAD Plan

## Topic
AI-supported lesson planning in TVET

## Build
Target group: TVET teachers and students  
Aim: Create one AI-supported lesson activity.

## Manage
Content sections:
- Introduction to AI lesson planning
- Prompt writing
- Human review
- Final activity creation

## Assess
Task: Create one AI-supported lesson activity and improve it manually.  
Evidence: AI prompt, first AI output, improved version, and reflection.

## Deliver
Online: GitHub repository  
Offline: Downloaded project folder or ZIP file
