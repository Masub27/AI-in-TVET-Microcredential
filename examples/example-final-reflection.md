# Example Final Reflection

In this project, I created an AI-supported online and offline teaching workflow package for TVET. The project includes documentation, learning modules, learner tasks, assessment strategy, templates, and quality checklists.

I used the BMAD Method to organize the project. First, I built the idea by defining the topic, target group, and aim. Then I managed the content by creating folders, modules, and tasks. After that, I created assessment criteria and evidence requirements. Finally, I prepared the project for online and offline delivery.

AI supported me by helping to create first drafts of learning materials, tasks, and assessment criteria. I reviewed the AI-generated content and improved it manually to make it clearer, more practical, and more suitable for TVET learners.

This project can support TVET learning because it gives learners a structured way to create digital teaching materials. It can be used online through GitHub and offline as a downloaded project folder.
